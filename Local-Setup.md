
## 2. Test CLI connection

### PowerShell

```powershell
irm https://codecrafters.io/install.ps1 | iex # Install CodeCrafters CLI
codecrafters ping
```

### Linux / macOS

```bash
curl -fsSL https://codecrafters.io/install.sh | bash # Install CodeCrafters CLI
```
